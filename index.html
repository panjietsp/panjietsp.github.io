Required rate of return

Discount rate

折现率是指将未来有限期预期收益折算成现值的比率. 在金融学上，只有同一个时间点的现金流才可以直接相加. 不同时间点的钱若要相加，就需要引入利率的概念，将所有的现金流折现到同一时间点(一般是今天).

举个例子:

Assuming a discount rate of 10%, 
if you expect to receive $1,000 in one year, the present value would be $909.09:
=1000/(1+0.1)
if you expect to receive $1,000 in two years, the present value would be $826.45:
=1000/(1+0.1)2
Oppotunity cost

在经济学中，机会成本不是一个真正的成本，而表示以现有的成本做了最优方案的投资后而损失的次有方案投资的收益，因为你的成本只能投资一次.

在金融学中机会成本指代一项基准利率，以高于此利率的某一利率进行投资，投资者就损失了以此利率投资可能带来的收益.

思考题:

为什么同一项金融产品的远期交易和期货交易价格不同？在考虑其他因素都相同的情况下
Decompose required rate of return

Nominal risk-free rate = real risk-free rate + expected inflation rate

Attention: “expected”表示将来的通胀率，而不是今天的通胀率。很好理解，存钱的收益是在将来获得的，所以通胀的影响也是在将来

Required interest rate on a security= nominal risk-free rate + default risk premium + liquidity risk premium +maturity risk premium

违约风险溢价：借款人不按期还款的风险
流动性风险溢价：投资资产在出售时没有获得公允价值的风险
到期风险溢价：长期债券比短期债券的溢价高
